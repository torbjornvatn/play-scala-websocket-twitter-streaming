Hvorfor er Websockets så kult?

  1. Data kan for første gang aktivt sendes frem og tilbake mellom server og nettleser via én kanal
  2. Nettleserkode trenger ikke å simulere "push" med f.eks. long polling
  3. Lett å sette opp demo
    4.         <script>
          $(document).ready(function(){
            ws = new WebSocket("ws://localhost:8008/websocket");
            ws.onmessage = function(evt) { 
              $("#msg").empty();
              $("#msg").append(evt.data); 
            };
            ws.onclose = function() {};
            ws.onopen = function() {};
          });
        </script>
